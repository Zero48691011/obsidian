# DGX、HGX、SXM 详解

> 这三个词是 NVIDIA 数据中心 GPU 生态中的核心概念，经常出现在 AI 基础设施选型中。理解它们的关系，是理解 NVIDIA 企业级 AI 硬件体系的关键。

---

## 一、一句话总结

| 术语 | 全称 | 本质 | 一句话 |
|------|------|------|--------|
| **SXM** | Socket for eXtreme Multi-GPU | **物理形态** | GPU 的插座/连接器规格，决定 GPU 怎么装到主板上 |
| **HGX** | Hyperscale GPU Xceleration | **GPU 基板平台** | 8 颗 SXM GPU + NVLink 的集成基板，卖给 OEM 做服务器 |
| **DGX** | Deep GPU Xceleration | **整机系统** | NVIDIA 自己品牌的全集成 AI 服务器，开箱即用 |

**关系**：NVIDIA 先把 GPU 做成 **SXM** 规格的模组 → 再把 8 颗 SXM GPU 装到一张 **HGX** 基板上 → HGX 有两种去向：① 卖给 OEM（超微、戴尔、HPE 等）做他们自己的服务器；② NVIDIA 自己加上 CPU、存储、网络、散热，做成 **DGX** 整机。

---

## 二、SXM — GPU 的物理连接方式

### 2.1 是什么

SXM 是 NVIDIA 专为数据中心 GPU 设计的高密度 mezzanine（夹层）连接器规格。它**不是 PCIe 插卡**，而是一种直接把 GPU 模块"贴装"到主板上的方式。

### 2.2 为什么要用 SXM 而不是 PCIe

| 对比维度 | SXM | PCIe 卡 |
|----------|-----|---------|
| 安装方式 | 螺丝固定到基板，无金手指 | 插入 PCIe 插槽 |
| NVLink 带宽 | 原生支持，GPU 间直连 | 需 NVLink Bridge 桥接，带宽受限 |
| 散热能力 | 450W–1000W+（大散热器） | 约 300W（双槽风冷） |
| 密度 | 一块基板 4–8 颗 GPU | 受 PCIe 槽位限制 |
| 典型场景 | DGX / HGX / 超算 | 普通 GPU 服务器 / 工作站 |

### 2.3 SXM 代际演进

| 版本 | 架构 | 代表 GPU | 发布时间 |
|------|------|----------|----------|
| SXM / SXM2 | Pascal | P100 | 2016 |
| SXM2 / SXM3 | Volta | V100 16GB/32GB | 2017 |
| SXM4 | Ampere | A100 40GB/80GB | 2020 |
| SXM5 | Hopper | H100 / H200 | 2022 |
| SXM6 | Blackwell | B100 / B200 | 2024 |
| SXM7 | Rubin | R100 | 2026（预计） |

> 每一代 SXM 的物理尺寸、针脚定义、功耗上限都不同，不可混插。

## 三、HGX — GPU 基板平台

### 3.1 是什么

HGX 是 NVIDIA 卖给服务器 OEM 厂商（超微、戴尔、HPE、联想、浪潮等）的 **GPU 基板套件**。它包含：

- 8 颗 SXM GPU（已装好）
- NVLink 互联网络（GPU 间高速通信）
- NVSwitch 芯片（全互联带宽）
- 基板 + 散热方案

### 3.2 HGX 的定位

**HGX = 服务器的心脏，但不是一个完整的服务器。**

OEM 拿到 HGX 基板后，自己加上：
- CPU（Intel Xeon 或 AMD EPYC 或 NVIDIA Grace）
- 系统内存（DDR5）
- 存储（NVMe SSD）
- 网络（InfiniBand / 以太网）
- 电源、机箱、散热

然后以自己品牌对外销售（如 "Supermicro SYS-821GE-TNHR" 搭载 HGX H100）。

### 3.3 当前 HGX 产品线

| 产品 | GPU 架构 | GPU 数量 | 定位 |
|------|---------|----------|------|
| HGX H100 | Hopper | 8× H100 SXM5 | 训练+推理 |
| HGX H200 | Hopper | 8× H200 SXM5 | 大模型推理（141GB HBM3e） |
| HGX B100 / B200 | Blackwell | 8× B100/B200 SXM6 | 下一代训练+推理 |
| HGX B200 Ultra | Blackwell Ultra | 8× | 更高 HBM 容量 |
| HGX Rubin NVL8 | Rubin | 8× R100 SXM7 | 2026，Agentic AI |

### 3.4 HGX 与 SuperPOD 的关系

HGX 是单节点（1 台服务器），多台 HGX 服务器 + NVIDIA 网络交换机可以组成 **DGX SuperPOD**（一个集群级 AI 超算）。

## 四、DGX — NVIDIA 整机品牌

### 4.1 是什么

DGX 是 NVIDIA 自己设计、自己品牌、自己销售和维护的 **全集成 AI 服务器**。可以理解为 "买回去插电就能训练大模型" 的完整产品。

每台 DGX 内部的核心就是一块 HGX 基板，但 NVIDIA 额外集成了：
- CPU、内存、存储
- 网络（InfiniBand / 以太网）
- BMC 管理 + 软件栈（NVIDIA AI Enterprise、Base Command、Mission Control）
- 全套散热、电源、机箱
- 企业级支持服务

### 4.2 DGX 代际演进

| 产品 | 架构 | GPU | 发布时间 | 首发价 |
|------|------|-----|----------|--------|
| DGX-1 | Pascal | 8× P100 | 2016 | $129,000 |
| DGX-1 (Volta) | Volta | 8× V100 | 2017 | $149,000 |
| DGX-2 | Volta | 16× V100 | 2018 | $399,000 |
| DGX A100 | Ampere | 8× A100 | 2020 | $199,000 |
| DGX H100 | Hopper | 8× H100 | 2022 | ~$482,000 |
| DGX GB200 | Blackwell | 36 Grace + 72 B200 | 2024 | — |
| DGX Station | Blackwell | 1× GB300 | 2026 | ~$100K–$120K |

### 4.3 DGX 的变体

| 变体 | 形态 | 说明 |
|------|------|------|
| **DGX Server** | 机架式 | 8U 机架服务器，数据中心主力 |
| **DGX Station** | 塔式工作站 | 静音水冷，适合办公室/实验室 |
| **DGX SuperPOD** | 集群 | 多台 DGX + 网络交换机组成的超算 |
| **DGX Spark** | 桌面小盒子 | GB10 芯片，128GB 统一内存，面向开发者 |

## 五、对比总结

### 5.1 三者关系图

```
┌─────────────────────────────────────────────────────────┐
│                     DGX 整机系统                          │
│  ┌───────────────────────────────────────────────────┐  │
│  │                    HGX 基板                        │  │
│  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                │  │
│  │  │ SXM │ │ SXM │ │ SXM │ │ SXM │  ... ×8        │  │
│  │  │ GPU │ │ GPU │ │ GPU │ │ GPU │                │  │
│  │  └─────┘ └─────┘ └─────┘ └─────┘                │  │
│  │           NVSwitch + NVLink 互联                   │  │
│  └───────────────────────────────────────────────────┘  │
│  + CPU + 内存 + 存储 + 网络 + 散热 + 电源 + 软件         │
└─────────────────────────────────────────────────────────┘
```

### 5.2 核心差异

| 维度 | SXM | HGX | DGX |
|------|-----|-----|-----|
| **层级** | 物理连接器 | GPU 子系统 | 完整服务器 |
| **卖给谁** | （不零售） | OEM 服务器厂 | 终端用户 |
| **品牌** | NVIDIA | NVIDIA（内部组件） | NVIDIA |
| **包含什么** | 仅 GPU 模组 + 插座 | 8 GPU + NVLink + 基板 | 全部硬件 + 软件 + 支持 |
| **可定制** | 不可 | OEM 可加 CPU/存储 | 有限配置选项 |
| **采购渠道** | 不单独卖 | 通过 OEM 采购 | NVIDIA 直销 / 合作伙伴 |

### 5.3 选型建议

| 场景 | 推荐 | 原因 |
|------|------|------|
| 买来直接用，不想折腾 | **DGX** | NVIDIA 集成好，开箱即用，有官方支持 |
| 已有服务器供应商，需要灵活定制 | **HGX 服务器** | 超微/戴尔等 OEM 的 HGX 服务器，性价比高 |
| 自建 AI 超算集群 | **DGX SuperPOD** 或 **HGX + 自组** | 看是要交钥匙方案还是自主集成 |
| 小团队/个人开发者 | **DGX Spark / Station** | 桌面级，价格可接受 |
| 只是了解硬件规格 | 关注 **SXM 代际** | SXM5/SXM6/SXM7 决定 GPU 的带宽和功耗上限 |

## 六、常见误区

1. **"HGX 就是 DGX 的简写"** ❌ → HGX 和 DGX 是完全不同的产品线，一个是基板平台，一个是整机系统。
2. **"SXM 是 GPU 型号"** ❌ → SXM 是连接器规格，不是芯片型号。H100 有 SXM5 版也有 PCIe 版。
3. **"买了 DGX 就不需要网络了"** ❌ → DGX 自带单机内部的 NVLink 互联，但多机之间仍需要 InfiniBand 或以太网交换机。
4. **"HGX 比 DGX 便宜"** ⚠️ → 不一定，HGX 基板 + OEM 集成 + 自购 CPU/存储/软件/支持，总成本可能和 DGX 接近甚至更高。
