# Python 列表、元组、字典 详解

## 一、一句话概括

| 类型 | 一句话 | 写法 |
|------|--------|------|
| **列表 (list)** | 可变的**有序**序列，啥都能装 | `[1, 2, 3]` |
| **元组 (tuple)** | **不可变**的有序序列，定义后不能改 | `(1, 2, 3)` |
| **字典 (dict)** | **键值对**的集合，通过 key 查 value | `{"name": "Tom", "age": 20}` |

---

## 二、横向对比

| 特性 | 列表 (list) | 元组 (tuple) | 字典 (dict) |
|------|-------------|-------------|------------|
| **可变性** | ✅ 可变（增删改） | ❌ 不可变（定死） | ✅ 可变（增删改） |
| **有序性** | ✅ 有序（下标访问） | ✅ 有序（下标访问） | ✅ 有序（Python 3.7+） |
| **访问方式** | 下标 `lst[0]` | 下标 `tup[0]` | 键 `d["key"]` |
| **是否可重复** | ✅ 元素可重复 | ✅ 元素可重复 | Key 不可重复，Value 可重复 |
| **性能** | 增删末尾 O(1) | 查找 O(1) | 查找 O(1) |
| **内存占用** | 较大 | 较小 | 较大 |
| **可哈希** | ❌ 不能做 dict 的 key | ✅ 可以做 dict 的 key | ❌ 不能做 dict 的 key |

---

## 三、列表 (list) — 可变序列

### 3.1 创建方式

```python
# 空列表
a = []
b = list()

# 有内容的列表
nums = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]   # 啥类型都能装
nested = [[1, 2], [3, 4]]          # 列表套列表

# 从其他可迭代对象创建
chars = list("abc")   # ['a', 'b', 'c']
```

### 3.2 常用操作

```python
lst = [10, 20, 30]

# ——— 增 ———
lst.append(40)          # 末尾追加 → [10, 20, 30, 40]
lst.insert(1, 15)       # 指定位置插入 → [10, 15, 20, 30, 40]
lst.extend([50, 60])    # 扩展列表 → [10, 15, ..., 50, 60]

# ——— 删 ———
lst.pop()               # 删除末尾并返回 → 60
lst.pop(1)              # 删除下标 1 的元素
lst.remove(20)          # 删除第一个值为 20 的元素
del lst[0]              # 删除下标 0 的元素

# ——— 查 ———
lst[0]                  # 按下标访问
lst[-1]                 # 最后一个
lst[1:3]                # 切片 → 下标 1 到 2
len(lst)                # 元素个数
20 in lst               # 判断是否存在 → True/False

# ——— 遍历 ———
for item in lst:
    print(item)

for i, item in enumerate(lst):   # 带下标
    print(i, item)
```

### 3.3 什么时候用列表

- 数据需要**频繁增删改**
- 需要**保持顺序**，按位置访问
- 同类型数据集合（如所有用户、所有分数）

---

## 四、元组 (tuple) — 不可变序列

### 4.1 创建方式

```python
# 空元组
a = ()
b = tuple()

# 有内容的元组
point = (3, 4)
person = ("Tom", 20, "Beijing")

# ⚠️ 单元素元组必须加逗号！
single = (1,)       # 这才是元组
not_tuple = (1)     # 这是 int，不是元组！

# 不加括号也行
t = 1, 2, 3         # 等同于 (1, 2, 3)
```

### 4.2 常用操作

```python
t = (10, 20, 30, 20)

# ——— 查 ———
t[0]          # 10
t[-1]         # 30
t[1:3]        # (20, 30)
len(t)        # 4
t.count(20)   # 2（统计出现次数）
t.index(30)   # 2（找到值的下标）

# ——— 遍历 ———
for item in t:
    print(item)

# ——— 不能做 ———
# t[0] = 100     ❌ 会报错！元组不可变
# t.append(40)   ❌ 没有 append 方法
```

### 4.3 解包 (unpacking)

```python
point = (3, 4)
x, y = point          # x=3, y=4

a, b, c = (1, 2, 3)  # 一一对应

# 用 * 接收剩余
first, *rest = (1, 2, 3, 4)   # first=1, rest=[2,3,4]
```

### 4.4 什么时候用元组

- 数据**不希望被修改**（如坐标、常量、配置）
- 函数返回多个值：`return x, y` 实际上返回的是一个元组
- 需要做**字典的 key**（因为元组不可变，可哈希）

---

## 五、字典 (dict) — 键值对

### 5.1 创建方式

```python
# 空字典
d = {}
d = dict()

# 有内容的字典
person = {"name": "Tom", "age": 20, "city": "Beijing"}

# 用 dict() 创建
d = dict(name="Tom", age=20)   # {"name": "Tom", "age": 20}

# 从元组列表创建
d = dict([("a", 1), ("b", 2)])   # {"a": 1, "b": 2}
```

### 5.2 常用操作

```python
d = {"name": "Tom", "age": 20}

# ——— 增 / 改 ———
d["city"] = "Beijing"        # 新增键值对
d["age"] = 21                # 修改已有 key 的值

# ——— 删 ———
d.pop("age")                 # 删除并返回 value
del d["name"]                # 直接删除
d.clear()                    # 清空整个字典

# ——— 查 ———
d["name"]                    # 直接取值（key 不存在会报错）
d.get("name")                # 安全取值（不存在返回 None）
d.get("phone", "N/A")        # 不存在时返回默认值 "N/A"
"name" in d                  # 判断 key 是否存在

# ——— 遍历 ———
for key in d:                # 遍历 key
    print(key, d[key])

for key, value in d.items():  # 同时遍历 key 和 value
    print(key, value)

for value in d.values():      # 只遍历 value
    print(value)

# ——— 获取所有 key / value ———
d.keys()      # dict_keys(['name', 'age'])
d.values()    # dict_values(['Tom', 20])
d.items()     # dict_items([('name', 'Tom'), ('age', 20)])
```

### 5.3 字典推导式

```python
# 快速生成字典
squares = {x: x**2 for x in range(5)}   # {0:0, 1:1, 2:4, 3:9, 4:16}

# 翻转 key-value
flipped = {v: k for k, v in d.items()}

# 过滤
filtered = {k: v for k, v in d.items() if v > 10}
```

### 5.4 什么时候用字典

- 需要通过**名字 / 标签**来查找数据（而非位置）
- 数据之间有**映射关系**（如：用户名 → 用户信息）
- 需要快速查找（O(1) 时间复杂度）

## 六、互相转换

```python
# 列表 → 元组
tuple([1, 2, 3])           # (1, 2, 3)

# 元组 → 列表
list((1, 2, 3))            # [1, 2, 3]

# 列表/元组 → 字典（必须是成对数据）
dict([("a", 1), ("b", 2)])  # {"a": 1, "b": 2}

# 字典 → 列表（只取 key 或 value）
list({"a": 1, "b": 2})          # ["a", "b"]
list({"a": 1, "b": 2}.values()) # [1, 2]
list({"a": 1, "b": 2}.items())  # [("a", 1), ("b", 2)]
```

## 七、常见坑

### 坑 1：列表做字典 key 会报错

```python
# ❌ 错误
d = {[1, 2]: "value"}   # TypeError: unhashable type: 'list'

# ✅ 正确：用元组
d = {(1, 2): "value"}
```

### 坑 2：修改遍历中的列表

```python
# ❌ 错误：边遍历边删除，会跳过元素
lst = [1, 2, 3, 4]
for item in lst:
    lst.remove(item)

# ✅ 正确：遍历副本
for item in lst[:]:
    lst.remove(item)
```

### 坑 3：字典取不存在的 key

```python
d = {"a": 1}
# d["b"]          # ❌ KeyError

d.get("b")        # ✅ 返回 None
d.get("b", 0)     # ✅ 返回默认值 0
```

### 坑 4："可变" 的元组

```python
t = (1, [2, 3], 4)
t[1].append(5)       # ✅ 不报错！因为改的是元组里那个列表
print(t)             # (1, [2, 3, 5], 4)

# t[0] = 10          # ❌ 这个才报错，直接改元组元素不行
```

> 元组的"不可变"指的是**引用不可变**，但引用的对象如果是可变的（如列表），那个对象本身可以变。

## 八、速查：选哪个？

| 场景 | 选择 |
|------|------|
| 存一组同类型数据，需要增删改 | **列表** |
| 存一组不希望被修改的数据 | **元组** |
| 通过名字查找数据，一对一映射 | **字典** |
| 函数返回多个值 | **元组**（自动解包） |
| 需要做字典的 key | **元组**（可哈希） |
| 需要快速判断是否存在 | **字典** 或 **集合** |

---

## 九、一句话总结

> **列表**是"能改的排队"，**元组**是"不能改的排队"，**字典**是"贴标签的储物柜"。
